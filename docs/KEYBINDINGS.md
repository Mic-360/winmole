# Keybindings

## Global
- `Ctrl+P`: open command palette
- `Tab`: switch between sidebar and content
- `Shift+Tab`: move focus back
- `?`: open help screen
- `q`: quit when focused on the sidebar

## Navigation
- `↑` / `k`: move up
- `↓` / `j`: move down
- `←` / `h`: previous action tab or panel
- `→` / `l`: next action tab or panel
- `Enter`: open selected screen or drill into a list item
- `Esc` / `Backspace`: leave the current drill-down or close modal

## Lists
- `/`: start list filtering or open log search
- `Space`: toggle selection
- `a`: select all items in the current selectable list
- `n`: clear all selections in the current selectable list
- `x`: execute the selected workflow or purge set
- `r`: refresh the active screen inventory

## Logs
- `p`: pause or resume tail behavior
- `PgUp` / `PgDn`: page through help or long log content

## Settings
- `Enter`: edit the selected setting
- `Space`: toggle boolean settings
